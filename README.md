# MERN Stack CRUD Blog

## What is it

This is simple blog made with the MERN stack (MongoDB, Express, React and NodeJS).  
The example features a CRUD app with both server-side and frontend in one repository.  
I made the project following
[this tutorial by Kyle Cook on youtube](https://www.youtube.com/watch?v=1NrHkjlWVhM), check it out.  
Feel free to use this code as you please.

## Installation

To run this app simply run

```
npm i
```

and then:

```
npm run devStart
```
